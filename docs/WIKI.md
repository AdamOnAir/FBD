# FBD

FBD is a BBS based on Undertale Universe. It can also be used as a template for other game universe.

## Installation

Download from GitHub with GIT

```bash
git clone https://github.com/EvrestRGB/FBD.git
```

OR with GitHub CLI

```bash
gh repo clone EvrestRGB/FBD
```

## Commands
