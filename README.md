## Compile and Run

Compile with GCC:
```
git clone https://gitHub.com/AdamOnAir/FBD.git
cd FBD
gcc -o fbd main.c
```

Compile with CMake:
```
git clone https://gitHub.com/AdamOnAir/FBD.git
cd FBD
mkdir build && cd build
cmake ..
make
```

Precompiled MS Windows executabes can be downloaded from [the Releases section.](https://github.com/FBDev64/FBD/releases)
